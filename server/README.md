## Environments
환경변수 설정을 위해 env 파일이 필요
.env.sample 과 .env.dev.sample 파일을 참고하여 설정

- Devlopment Mode : .env.dev
- Production Mode : .env

## Run

## Devlopment Mode

### Run

```
> ./run-dev.sh up
```

### Stop

```
> ./run-dev.sh down
```

## Production Mode

```
> ./run.sh up
```

### Stop

```
> ./run.sh down
```
